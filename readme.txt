Import Images via Bing search API

== Description ==

BingImport provides you with the ability to import images from Bing. You can search for keywords, rename files and import the files. BingImport will attach the imported file to the post/page and can also be found in the Media Library for future use.

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/bingImport` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for bingimport via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin

Then in order to be able to use the plugin you need to have an API key. For this go to https://datamarket.azure.com/dataset/5BA839F1-12CE-4CCE-BF57-A49D98D29A44 and register and sign up for the free package (right at the bottom). If you sign in and go to https://datamarket.azure.com/account/info you will be able to get your Primary Account Key which you need to add to the plugin settings.

== Frequently Asked Questions ==


= Why do I need this plugin? =

If you usually search for images in order to download them and then upload them again to your wordpress site, this plugin wil save you loads of time.
